# Thomas Portfolio

A personal portfolio site built with **Vue 3**, **Vite**, and **Tailwind CSS v4**.

## Stack

- [Vue 3](https://vuejs.org/) — Composition API with `<script setup>`
- [Vite](https://vite.dev/) — Fast dev server and build tool
- [Tailwind CSS v4](https://tailwindcss.com/) — Utility-first styling via `@tailwindcss/vite`

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start development server |
| `npm run build`   | Production build         |
| `npm run preview` | Preview production build |

## Project structure

```
src/
├── App.vue              # Root layout
├── main.js              # App entry
├── style.css            # Tailwind + theme tokens
└── components/
    └── HeroSection.vue  # Landing hero (starter)
```
